# SchoolGradebook

A simple and efficient web application designed to help teachers manage student grades and attendance in one place. Developed as an individual project using ASP.NET Core MVC.

## 🚀 Features
* **Student Management:** Keep track of all your students.
* **Grade Tracking:** Easily add and edit grades for different subjects.
* **Attendance System:** Log daily attendance to monitor student progress.

## 🛠️ Tech Stack
* **Framework:** ASP.NET Core MVC
* **Language:** C#
* **Database:** SQLite
* **ORM:** Entity Framework Core
* **Frontend:** Bootstrap, HTML5, CSS3

## 📦 How to Setup
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/codingburgas/SchoolGradebook.git](https://github.com/codingburgas/SchoolGradebook.git)
   ```
2. **Open in your IDE**
3. **Setup Database:** Open the terminal in your editor and run:
   ```bash
   dotnet ef database update
   ```
4. **Run the app:** Click the "Run" button in your IDE.
